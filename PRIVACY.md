# MonoFocus Privacy Policy

Effective date: 2026-06-11

MonoFocus works locally on your device.

MonoFocus stores only the package names of apps you select. This data is stored in the app's private local storage and is not uploaded by MonoFocus.

MonoFocus uses Android Usage Access only to detect which app is currently open. It does not read the contents of other apps.

MonoFocus does not read notifications, messages, text, images, browser history, screen content, or files from other apps.

MonoFocus does not transmit data to any server.

MonoFocus does not use analytics, advertising SDKs, remote configuration, accounts, cloud sync, or crash-reporting SDKs.

MonoFocus does not sell or share user data.

Required Android accesses:

- Usage Access: used only to detect the foreground app package.
- Modes / Do Not Disturb access: used only to manage MonoFocus's own automatic rule for Android's grayscale display effect. MonoFocus configures this rule to allow notifications.

MonoFocus requires Android 15 or newer because it uses Android's public grayscale device effect API.
